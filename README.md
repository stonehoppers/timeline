# Adding events

if you want to contribute to the events list, follow this pattern for each event

```
{
  "name":"event name",
  "start":"yyyy/mm/dd hh:mm:ss GMT+0",
  "end":"yyyy/mm/dd hh:mm:ss GMT+0",
  "game":""
}
```

allowed games are "genshin", "hsr", "zzz", "hi3"
